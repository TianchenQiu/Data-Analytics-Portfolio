
# Marketing A/B Testing Analysis

# Project Overview

This project analyzes a marketing A/B test from Kaggle to evaluate whether an ad campaign increased user conversion compared with a PSA control group.

# Business Questions 

1. Would the campaign be successful?
2. If the campaign was successful, how much of that success could be attributed to the ads?

# Data Source

Dataset: Kaggle Marketing A/B Testing  
Link: https://www.kaggle.com/datasets/faviovaz/marketing-ab-testing
